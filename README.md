# GMod Система Кулдаунов

Продвинутая система кулдаунов для Garry's Mod с ограничением спавна ентити и современным меню управления.

## Установка

1. Скачайте все файлы из репозитория
2. Поместите все Lua файлы в папку `lua/autorun/` вашего сервера
3. Перезапустите сервер или смените карту

### Структура файлов
lua/autorun/
├── sv_cooldown.lua # Серверная система кулдаунов
├── cl_menu.lua # Админ-меню
└── cl_notifications.lua # Клиентские уведомления

## Использование

### Для Админов
1. Зайдите на сервер с правами админа
2. Откройте консоль и введите: `kd_menu`
3. Используйте меню для управления кулдаунами

**Добавление кулдауна:**
- Введите название ентити (например: `prop_physics`, `weapon_pistol`)
- Установите время кулдауна в секундах
- Нажмите "ДОБАВИТЬ"

**Удаление кулдауна:**
- Найдите ентити в списке
- Нажмите "УДАЛИТЬ" рядом с ним

### Для Игроков
- Пытайтесь спавнить ентити как обычно
- Получайте уведомления когда кулдаун активен
- Ждите окончания кулдауна перед следующим спавном

## Поддерживаемые типы ентити

Система работает со всеми основными типами спавна:
- Пропы (`PlayerSpawnProp`)
- SENTы (`PlayerSpawnSENT`)
- NPC (`PlayerSpawnNPC`)
- Рэгдоллы (`PlayerSpawnRagdoll`)
- Оружие (`PlayerSpawnSWEP`, `PlayerGiveSWEP`)

## Команды

- `kd_menu` - Открыть админ-меню управления кулдаунами

## Функции защиты

### Система кулдаунов
- Предотвращает быстрый спавн ентити
- Индивидуальные кулдауны для каждого игрока
- Визуальный отсчет времени

### Защита от обхода
- Блокирует спавн ентити через дубликатор
- Работает со всеми методами спавна
- Автоматическое обновление ограничений

## Визуальные особенности

### Админ-меню
- Темная тема с неоновыми синими акцентами
- Плавные анимации при наведении
- Крупный и удобный интерфейс
- Обновление списка в реальном времени

### Уведомления
- Анимация появления справа
- Прогресс-бары с оставшимся временем
- Цветные сообщения (синий=начало, красный=активно, зеленый=конец)
- Система очереди для нескольких уведомлений

## Требования

- Сервер Garry's Mod
- Поддержка SQLite (включена по умолчанию)

## Решение проблем

**Меню не открывается:**
- Проверьте права админа
- Проверьте консоль на ошибки Lua

**Кулдауны не работают:**
- Убедитесь что все файлы в `lua/autorun/`
- Проверьте консоль сервера на сообщения инициализации

**Ошибки SQL:**
- Проверьте права на запись у сервера
- Проверьте свободное место на диске
